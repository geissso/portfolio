# Portfolio Forum Alte - Astro

Site portfolio migre completement vers Astro + Tailwind + CSS + JavaScript natif.

## Lancer en local

1. Installer les dependances

```bash
npm install
```

2. Demarrer le serveur local

```bash
npm run dev
```

Le site est disponible sur http://localhost:8080

## Scripts utiles

```bash
npm run dev      # serveur de dev
npm run build    # build production
npm run preview  # previsualiser le build
npm run check    # verification Astro
```
